  ## 6. 담당 기능
  - javascript와 jquery를 사용하여 검색 키워드를 검색결과 창으로 전달
  - 음식 종류별로 간편하게 모아서 찾아볼 수 있도록 음식 카테고리를 구현
  - Swiper 라이브러리를 사용하여 평점 순으로 인기 있는 맛집을 슬라이더 형식으로 구현
  - javascript를 통해 아이디와 비밀번호 유효성 검사 작업
  - Http 쿠키를 활용하여 로그인 정보 저장 가능
